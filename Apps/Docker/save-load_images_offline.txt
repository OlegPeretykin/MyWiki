Оффлайн-использование: После скачивания образ будет храниться локально, и вы сможете использовать его без интернета с помощью docker run, указав тот же тег, который скачали.

Сохранение образа: Если вы хотите сохранить образ в файл для переноса на другую машину, выполните:
bash

docker save -o minio.tar quay.io/minio/minio

Затем перенесите файл minio.tar и загрузите его на другой машине:
bash

docker load -i minio.tar
