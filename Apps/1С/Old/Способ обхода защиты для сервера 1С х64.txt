Способ обхода защиты для сервера 1С х64 (не путать с Windows Server х64!)
 
 1. Включаем тестовый режим Windows Server (через cmd или PowerShell): bcdedit.exe -set TESTSIGNING ON
 2. Перезагружаемся
 3. Добавляем дампы (все три) в реестр
 4. РУКАМИ, через Диспетчер устройств, добавляем старое оборудование и указываем на драйвер HASP эмулятора
 5. Устанавливаем HASP License Manager для доступа в терминале
 6. Перезагружаемся
 
PS: Не забываем открыть порты на сервере, как для 1С. так и для MS SQL
 
PPS: Лицензия на 500 пользователей - она от лукавого, это лицензия на 50 пользователей, "пропатченная" на 500
