Руководство по xdotool: Автоматизация действий в X11
Введение

xdotool - это инструмент командной строки для автоматизации действий в X11 (система X Window System). Он позволяет программно или вручную имитировать ввод с клавиатуры и мыши, перемещать и изменять размеры окон, а также выполнять другие операции, связанные с графическим интерфейсом пользователя.

Установка

xdotool доступен для большинства дистрибутивов Linux. Вы можете установить его с помощью следующей команды:

sudo apt install xdotool
Основные команды

key [options] keystroke [keystroke ...]: Симулирует нажатие клавиш.
-window: Указывает окно, на которое будут отправлены нажатия клавиш.
--clearmodifiers: Сбрасывает все модификаторы клавиш (Shift, Ctrl, Alt, Super) перед отправкой нажатия.
--delay: Задерживает отправку нажатия клавиш на указанное количество миллисекунд.
mouse [options] x y [click button]: Симулирует движение мыши и щелчки.
-window: Указывает окно, в котором будет происходить движение мыши.
x, y: Координаты X и Y на экране.
click: Указывает тип щелчка (1 - левая кнопка, 2 - средняя кнопка, 3 - правая кнопка).
button: Указывает кнопку мыши (1 - левая, 2 - средняя, 3 - правая).
windowfocus [options] window_id: Переключает фокус на заданное окно.
windowactivate [options] window_id: Активирует заданное окно.
windowmove [options] window_id x y: Перемещает заданное окно в указанную позицию.
windowresize [options] window_id width height: Изменяет размер заданного окна.
windowkill [options] window_id: Закрывает заданное окно.
Примеры использования

Нажать клавишу Enter:
xdotool key Enter
Нажать клавишу Ctrl+C:
xdotool key --clearmodifiers Ctrl+c
Переместить курсор мыши в точку (100, 200) и щелкнуть левой кнопкой:
xdotool mouse 100 200 click 1
Переключить фокус на окно с ID 12345:
xdotool windowfocus 12345
Активировать окно с ID 12345:
xdotool windowactivate 12345
Переместить окно с ID 12345 в позицию (50, 50):
xdotool windowmove 12345 50 50
Изменить размер окна с ID 12345 на 300x200:
xdotool windowresize 12345 300 200
Закрыть окно с ID 12345:
xdotool windowkill 12345
Поиск окон

xdotool позволяет искать окна по различным критериям, таким как название, класс, ID, PID и т.д.

Поиск по названию:
xdotool search --name "Firefox"
Поиск по классу:
xdotool search --class "Gimp"
Поиск по ID:
xdotool search --window 12345
Поиск по PID:
xdotool search --pid 12345
Получение информации о окнах

xdotool может предоставить информацию о окнах, таких как название, класс, ID, PID, геометрия и т.д.

Получить информацию о текущем активном окне:
xdotool getactivewindow
Получить информацию о окне с ID 12345:
xdotool windowinfo 12345
