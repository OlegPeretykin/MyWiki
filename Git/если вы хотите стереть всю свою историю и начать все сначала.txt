если вы хотите стереть всю свою историю и начать все сначала:


если у вас есть только локальный репозиторий git
cd <repo>
rm -rf .git
git init
и снова начинайте совершать.

Если вы хотите удалить как файлы, так и историю:
cd <repo>
rm -rf *
git init
и начните добавлять файлы и совершать фиксацию...

если вы связаны с удаленным хранилищем , если вы хотите начать все сначала, но на самом деле не возражаете, чтобы осталась какая-то старая история; есть быстрый способ:
git pull
git rm -r *
git commit
git push
теперь ваш репозиторий снова пуст, у вас останется только старая история. если вы также хотите очистить всю свою историю; 

это займет немного больше работы (и обратите внимание, что это вызовет проблемы для всех, кто связан с тем же удаленным хранилищем):
git checkout <first commit hash>
git rm -r *
touch README
git add README
git commit --amend
git push -f
обратите внимание, что я создаю пустой файл README для первой фиксации(коммита), так как фиксация(коммит) не может быть пустой.

если вы говорите о существующем удаленном репо (а не только о локальном репо, что тривиально), вы можете:
•	клонировать его
•	удалить все удаленные ветви: git push origin --delete <branchName> (см. " Удалить Git branch как локально, так и удаленно ")
•	создайте новую сиротскую главную ветвь (см. " Как я могу полностью очистить главную ветвь в Git? ")
•	git branch -D master
git checkout --orphan master
•	сделайте хотя бы один коммит и нажмите на него.
