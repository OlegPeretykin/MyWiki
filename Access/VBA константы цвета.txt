VBA константы цвета
В операторах и функциях VBA для определения цветовой палитры можно использовать следующие константы

Константа					Значение	Описание переменной
vbBlack						x0			Черный
vbRed						0xFF		Красный
vbGreen						0xFF00		Зеленый
vbYellow					0xFFFF		Желтый
vbBlue						0xFF0000	Синий
vbMagenta					0xFF00FF	Розовый
vbCyan						0xFFFF00	Голубой
vbWhite						0xFFFFFF	Белый


Следующие константы могут использоваться в любом месте программы вместо их значений:
Константа					Значение	Описание переменной
vbScrollBars				0x80000000	Цвет полосы прокрутки
vbDesktop					0x80000001	Цвет рабочего стола
vbActiveTitleBar			0x80000002	Цвет заголовка активного окна
vbInactiveTitleBar			0x80000003	Цвет заголовка неактивного окна
vbMenuBar					0x80000004	Цвет фона меню
vbWindowBackground			0x80000005	Цвет фона окна
vbWindowFrame				0x80000006	Цвет рамки окна
vbMenuText					0x80000007	Цвет текста меню
vbWindowText				0x80000008	Цвет текста окна
vbTitleBarText				0x80000009	Цвет кнопок управления окном
vbActiveBorder				0x8000000A	Цвет границы активного окна
vbInactiveBorder			0x8000000B	Цвет границы неактивного окна
vbApplicationWorkspace		0x8000000C	Цвет фона для приложений интерфейса MDI
vbHighlight					0x8000000D	Цвет фона выбранных элементов управления
vbHighlightText				0x8000000E	Цвет текста выбранных элементов управления
vbButtonFace				0x8000000F	Цвет затенения для передней части кнопок управления
vbButtonShadow				0x80000010	Цвет затенения для края кнопок управления
vbGrayText					0x80000011	Серый (неактивный) текст
vbButtonText				0x80000012	Цвет текста кнопок
vbInactiveCaptionText		0x80000013	Цвет текста активной подписи
vb3DHighlight				0x80000014	Цвет выделения для трехмерных элементов
vb3DDKShadow				0x80000015	Цвет затенения для трехмерных элементов
vb3DLight					0x80000016	Второй светлый цвет для трехмерных элементов после vb3Dhighlight
vbInfoText					0×80000017	Цвет текста всплывающей подсказки
vbInfoBackground			0×80000018	Цвет фона всплывающей подсказки
