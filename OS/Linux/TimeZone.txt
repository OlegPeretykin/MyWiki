Изменение часового пояса с помощью timedatectl Command

Чтобы изменить настройки часового пояса в вашем терминале, вам сначала нужно найти регион и формат города. Для этого вы будете использовать (timedatectl) команда для печати этого списка в (Регион / город) Формат.
timedatectl list-timezones

Как только вы найдете правильную запись, вы будете использовать (timedatectl) чтобы установить часовой пояс с помощью следующей команды:

sudo timedatectl set-timezone your_time_zone

Для учебника он был установлен на (Австралия / Квинсленд) используя команду:
Пример:
sudo timedatectl set-timezone Australia/Queensland
Затем проверьте, был ли применен часовой пояс, повторно введя (timedatectl) команда:

timedatectl
