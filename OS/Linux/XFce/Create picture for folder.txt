Чтобы изменить иконку конкретной папки в среде рабочего стола XfCE на Debian 12, выполните следующие шаги:

Шаг 1: Найдите иконку

Выберите файл PNG или SVG формата, который хотите использовать в качестве иконки для вашей папки. Поместите этот файл в доступное место, чтобы легко было указать путь к нему позже.

### Шаг 2: Создайте файл .desktop

Для изменения иконки папки вам нужно создать специальный файл .desktop внутри самой папки. Откройте терминал и перейдите в директорию, где находится ваша папка:

cd /путь/до/вашей/папки


Затем создайте файл с именем folder-name.desktop, заменив folder-name на название вашей папки. Например, если ваша папка называется Документы, создайте файл Документы.desktop.

Откройте этот файл в любом текстовом редакторе (например, nano):

nano Документы.desktop


Добавьте следующий контент в файл:

[Desktop Entry]
Name=Название Папки
Icon=/путь/до/иконки.png
Type=Directory


Замените:
- Название Папки на имя вашей папки.
- /путь/до/иконки.png на полный путь до вашего файла иконки.

Например:

[Desktop Entry]
Name=Документы
Icon=/home/user/images/my-icon.png
Type=Directory


После внесения изменений сохраните файл (Ctrl + O) и закройте редактор (Ctrl + X).

Шаг 3: Обновление иконок

Теперь перезапустите рабочий стол XFCE, чтобы изменения вступили в силу. Это можно сделать через команду:

xfdesktop --reload


Или просто перезапустить систему.

Альтернативный способ

Если вы предпочитаете графический интерфейс, попробуйте следующее:

1. Щелкните правой кнопкой мыши на значке папки.
2. Выберите пункт Свойства (или аналогичное меню).
3. В открывшемся окне найдите параметр для выбора иконки.
4. Укажите путь к вашему изображению.
