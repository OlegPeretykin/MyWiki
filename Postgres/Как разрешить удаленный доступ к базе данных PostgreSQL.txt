Отредактируйте файл pg_hba.conf и добавьте следующую запись в самом конце файла:

host    all             all              0.0.0.0/0                       md5
host    all             all              ::/0                            md5

------------------------------

host all all 192.168.56.1/24 md5
таким образом, PostgreSQL будет принимать соединения от всех хостов в сети 192.168.1.XXX.
